# Documentation

## Getting started

This documentation is a Single Page Application. Opening the index.html in your browser will not work. To use this documentation please follow this instruction

- First, ensure that node.js & npm are both installed. If not, choose your OS and installation method from [this page](https://nodejs.org/en/download/package-manager/) and follow the instructions.
- You need to install a simple static web serve called serve. To do this run `npm install serve -g` or if you are using yarn `yarn global add serve`
- Next, use your command line to enter into this directory.
- Run `serve -s build`
- Paste the printed url in your browser and you should see the docs.
